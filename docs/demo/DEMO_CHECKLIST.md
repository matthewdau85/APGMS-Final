# APGMS Demo Checklist

Pre-demo:
- Start webapp
- Sign in as Admin
- Open APGMS Console (Demo Mode)
- Confirm Setup Wizard completed (or run it quickly)
- Set period to 2025-Q1
- Simulation OFF initially

During demo:
- Dashboard: turn Simulation ON and wait for 1 feed event
- Obligations: Run reconciliation -> Prepare lodgment -> Submit -> Queue payment -> Generate evidence pack
- Ledger: filter by obligation
- Evidence Pack: open latest pack
- Controls: toggle an export default
- Incidents: create one incident
- Settings: show cadence + simulation frequency + reset control
- Regulator Portal: show read-only summary

Post-demo:
- Turn Simulation OFF
- Optionally Reset demo state
