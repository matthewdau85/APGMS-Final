# Security exceptions register

Purpose:
- Track time-boxed exceptions to security controls with explicit risk acceptance.

Required fields for each exception:
- Title
- Control being excepted
- Justification
- Risk and impact
- Compensating controls
- Owner
- Approved by
- Created date
- Expiry date (mandatory)
- Review date

Template:
Create files in this folder named:
  EXC-YYYYMMDD-<short-title>.md
