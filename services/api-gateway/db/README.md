\# Prisma schema location



Do not add a schema.prisma in this directory.



Canonical Prisma schema + migrations live at:



\- infra/prisma/schema.prisma

\- infra/prisma/migrations/



Run Prisma via workspace root scripts, e.g.:



\- pnpm db:gen

\- pnpm db:migrate

\- pnpm db:deploy

\- pnpm prisma:status
